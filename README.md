# “奇点升学”考研小程序README文档

## 项目概述

“奇点升学” 考研小程序是一个专为考研学生打造的综合性服务平台，提供院校信息查询、考研资料分享、反馈提交等功能，助力考生高效备考。

## 配置须知

### Node 版本要求

在运行之前，请确保自己的node版本不高于18；如若不符合要求，请使用 `nvm` 工具切换版本：

```PowerShell
nvm use 18.xx.x
```

### 微信 npm 安装和构建

在运行之前，请点击“工具” - “构建npm”进行微信npm的安装和构建，完成后会在项目目录下生成一个 `miniprogram_npm` 文件夹。
